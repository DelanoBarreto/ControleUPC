# Índice dos documentos para Codex

Coloque esta pasta dentro do seu projeto, por exemplo:

```txt
/docs/codex/
```

Arquivos:

1. `README-IA.md`  
   Contexto geral do projeto para o Codex.

2. `ARQUITETURA-SUPABASE-UNICO.md`  
   Explica a arquitetura com 1 VPS, 1 Supabase e vários produtos.

3. `PRD-PORTALGOV.md`  
   Documento técnico inicial do PortalGov.

4. `PRD-PRESTACAO-CONTAS.md`  
   Documento técnico inicial da Prestação de Contas.

5. `PROMPTS-CODEX.md`  
   Prompts prontos para usar no Codex.

6. `BANCO-MODELO-INICIAL.md`  
   Resumo prático do modelo inicial do banco.

Sugestão:
Sempre peça para o Codex ler primeiro `README-IA.md` e depois o PRD do produto em que você estiver trabalhando.
